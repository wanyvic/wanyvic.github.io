Rust笔记(一) 安装

发表于 2021-10-23 分类于 rust 阅读次数： Waline： 本文字数： 642 阅读时长 ≈ 1 分钟

安装

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

更新

rustup update

卸载

rustup self uninstall

编译

rustc xxx.rs

命名规范

snake_case: 文件名
CamelCase:

Cargo

cargo 是rust的构建系统和包管理工具。

创建项目

cargo new xxx # 创建二进制项目
cargo new xxx --lib # 创建库项目

安装依赖

cargo install xxxx

编译

cargo build
cargo build --release

运行

cargo run

tests

cargo test

检查

检查代码，cargo check比cargo build运行时间快的多

cargo check

panic 中断

Cargo.toml

[profile.release]
panic = 'abort'

[dependencies] 添加package
